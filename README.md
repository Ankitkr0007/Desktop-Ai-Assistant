# 🤖 Friday Assistant

Friday Assistant is an AI-powered virtual assistant built using Python. It is designed to automate everyday tasks through voice and text commands, making interactions faster, smarter, and more efficient.

---

## ✨ Features

- 🎤 Voice Recognition
- 💬 AI-Powered Chat
- 🌤️ Real-Time Weather Updates
- 📰 Latest News
- 🌐 Open Websites
- 📂 Launch Applications
- 🔊 Text-to-Speech Responses
- ⚡ Task Automation
- 🧠 Intelligent Command Processing

---

## 🛠️ Tech Stack

- Python
- SpeechRecognition

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Friday-Assistant.git
```

### 2. Navigate to the Project Directory

```bash
cd Friday-Assistant
```

### 3. Create a Virtual Environment

```bash
python -m venv fridayenv
```

### 4. Activate the Virtual Environment

#### Windows

```bash
fridayenv\Scripts\activate
```

#### Linux/macOS

```bash
source fridayenv/bin/activate
```

### 5. Install Required Packages

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

```bash
python main.py
```
